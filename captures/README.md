# SignalLab Capture Gallery

The base gallery is generated by `tools/capture_index.py` so captures can be inspected visually on GitHub.
Each graph is the raw 10-bit GPIO35 waveform from `preview.svg`; programmatic summaries are shown only as helpers.

Manual raw-signal quality ratings from [`docs/initial-capture-analysis.md`](../docs/initial-capture-analysis.md):

![best](https://img.shields.io/badge/rating-best-brightgreen) connected capture with sustained candidate waveform structure.
![maybe](https://img.shields.io/badge/rating-maybe-yellow) connected capture with candidate segments but material signal-quality caveats.
![control](https://img.shields.io/badge/rating-control-blue) no-contact/unplugged negative control or false-positive test.
![ignore](https://img.shields.io/badge/rating-ignore-lightgrey) not useful for waveform analysis, though it may remain useful as bad-signal data.

Regenerate after adding or editing captures:

```bash
python3 tools/capture_index.py
```
